# Catan Game Project (Gen3)

A modern Python + Pygame implementation of the classic board game **Settlers of Catan**.

![Screenshot](path-to-screenshot.png)

## Overview

This project is a highly modular and visually rich implementation of Catan with:
- Fully interactive human Initial Placement
- Multiple AI placement algorithms (including Markov Chain evaluation)
- Clean GUI with animations, pulsing highlights, and dynamic buttons

**[📄 Full Project Overview (PDF)](docs/Overview_v4.pdf)**

## Features

- 45-tile hexagonal board (19 land + 26 sea)
- Human + AI players
- Advanced Initial Placement strategies:
  - Max Pips
  - Max Pips + Ports
  - 5 Weighted Strategic Strategies
  - **Markov Chain** evaluator (most advanced)
